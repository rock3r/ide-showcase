## Navigate your code

* Go to...
  * Class: `Cmd + O`
  * File: `Cmd + Shift + O`
  * Symbol: `Cmd + Alt + O`
* Recent files: `Cmd + E`
* Recent locations: `Cmd + Shift + E`
* File Structure: `Cmd + F12`
  ![file structure](img/file-structure.png)
* Go to next error: `F2`
* Go to Declaration/usages: `Cmd + B`
* Go to Implementations: `Cmd + Alt + B`
* Go to Type declaration: `Cmd + Shift + B`
